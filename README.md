# redact-data
[![](http://meritbadge.herokuapp.com/redact-data)](https://crates.io/crates/redact-data) [![License: MIT](https://img.shields.io/badge/License-GPL3.0-yellow.svg)](https://opensource.org/licenses/GPL-3.0) [![Coverage Status](https://coveralls.io/repos/github/pauwels-labs/redact-data/badge.svg?branch=main)](https://coveralls.io/github/pauwels-labs/redact-data?branch=main)

Contains all data and data storage abstractions for the Redact framework.

Also contains implementations of the storage traits. Current supported implementations are:
- mongodb
- redact-store

To get started using this crate, view the docs [here](https://docs.rs/redact-data).
